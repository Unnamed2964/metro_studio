# components 目录说明

该目录存放 UI 组件，负责交互与渲染：

- `ToolbarControls.vue`
  - 工程管理（新建、保存、加载）
  - OSM 导入参数控制（在建/规划）
  - 编辑模式切换（选择/点站/拉线）
  - 自动排版触发
  - SVG/PNG 导出
- `MapEditor.vue`
  - 基于 MapLibre 的真实地图编辑器
  - 站点点击、拖拽、加点、连线
  - 渲染线路与站点图层
- `SchematicView.vue`
  - 渲染自动排版后的官方风示意图
  - 连线采用八向折线渲染（仅水平/垂直/45°对角）
  - 支持滚轮缩放与中键平移浏览
  - 画布自适应容器铺满显示
  - 展示评分与快照信息
